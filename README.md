# 🏪 AI 智能门店选址分析系统

基于高德地图 API 和 Python Streamlit 构建的门店选址智能分析工具。输入候选地址，自动查询周边竞品、交通、客流等数据，按照自定义评分标准打分排名，输出推荐结果。

## 📌 项目背景

传统门店选址需要人工实地调研、手动记录数据、逐项对比分析，效率低且容易遗漏。本项目利用高德地图开放 API 获取实时地理信息，结合可配置的评分算法，实现选址分析的自动化和数据化，帮助创业者快速评估多个候选位置的优劣。

## 🚀 核心功能

- **地址批量解析**：支持同时输入多个候选地址，自动转换为经纬度坐标。
- **周边 POI 查询**：调用高德地图 API，查询 500 米范围内的：
  - 同类竞品（如奶茶店、便利店等）数量
  - 最近地铁站距离
  - 写字楼、学校数量
  - 餐饮店、便利店数量
- **多维度自动评分**：按照可配置的评分规则（竞品密度 30 分、交通便利 25 分、客流来源 25 分、周边配套 20 分），对每个地址逐项打分。
- **结果排名与推荐**：以表格形式清晰展示各地址的分项得分、总分和评级（强烈推荐 / 推荐 / 一般 / 不推荐），并给出综合推荐建议。

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| Python 3 | 核心开发语言 |
| Streamlit | 快速构建数据可视化 Web 界面 |
| 高德地图 Web API | 地理编码、周边 POI 检索 |
| Pandas | 数据处理与表格展示 |
| Dotenv | 环境变量管理（保护 API 密钥） |
